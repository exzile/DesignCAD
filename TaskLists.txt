P3 — MISSING SKETCH MODIFY TOOLS
[x] D16. Sketch Fillet — pick corner where two lines meet + radius input.
         - Shipped: 'sketch-fillet' Tool type added. Click near a shared vertex
           of two line entities; bisector-geometry computes the arc center and
           tangent-point offsets. Both line endpoints are trimmed back to the
           tangent points; a new 'arc' entity is inserted. Fillet radius exposed
           as `sketchFilletRadius` in cadStore (default 2) + Radius input in
           SketchPalette while the tool is active. Toolbar Fillet button wired.

[x] D17. Trim — click a line segment portion to delete it up to nearest
         intersections.
         - Shipped: 'trim' Tool type added. Click near a line → compute all
           2D line-line intersections (plane-local coords via t1/t2) → remove
           the sub-segment between the two nearest intersection t-values.
           Endpoints outside the trimmed interval become new shorter entities.
           Uses replaceSketchEntities. Toolbar Trim button wired.

[x] D18. Extend — click near an endpoint, extends to nearest intersection.
         - Shipped: 'extend' Tool type added. Click near a line endpoint (within

           extended ray intersects any other line in the sketch. Updates the
           endpoint to the intersection point. Toolbar wired.

[x] D19. Break — click on a line; splits it into two entities at that point.
         - Shipped: 'break' Tool type added. Closest line entity to the click

P7 — SKETCH CREATE (variants we missed on the first pass)
[x] D40. 2-Tangent Circle
         - Shipped: 'circle-2tangent' Tool. Click 2 lines (nearest-to-click
           selection). Normalized 2D line equations (nu,nv,c) in sketch plane.
           4 candidate centers from ±r offsets on each line × line intersection.
           Picks candidate nearest to click midpoint. Radius from
           tangentCircleRadius in cadStore + SketchPalette input. Toolbar wired.

[x] D41. 3-Tangent Circle
         - Shipped: 'circle-3tangent' Tool. 3 clicks select 3 lines. Iterates
           all 8 sign combinations for the bisector-intersection system to find
           the smallest-radius real incircle. Emits circle entity. Toolbar wired.

[x] D42. Line-tool click-drag → tangent arc
         - Shipped: pointerdown/pointermove/pointerup state machine in
           SketchInteraction. When the user has a chain anchor (drawingPoints
           length > 0) in line mode and drags > 8px, isDraggingArcRef activates.
           On pointerup the tangent arc is computed (chord / 2·proj formula from
           arc-tangent tool) and committed; the arc end becomes the new chain
           anchor. The click event that follows is suppressed by dragJustFinished
           flag. useFrame shows live arc preview during drag.

[x] D43. Midpoint Line
         - Shipped: 'midpoint-line' Tool type. Click 1 = midpoint, click 2 =
           one endpoint. The opposite endpoint mirrors: otherPt = 2*midPt -
           endpoint. Emits a standard 'line' entity. Preview shows both
           endpoints + midpoint cross marker. Toolbar → Line submenu wired.

[x] D44. Blend Curve
         - Shipped: 'blend-curve' Tool. 2-click flow: click near endpoint of
           any line/arc/spline entity (within 6 world units), then click near
           a second endpoint. Tangent at each end is computed from the entity
           geometry (line direction, arc tangent at angle, spline first/last
           segment). A G1 cubic Bezier (P1 = P0 + h·tangentA, P2 = P3 - h·tangentB,
           h = |P3-P0|/3) is sampled at 32 points and emitted as 'spline'.
           Live preview shows the blended curve as the user moves toward the
           second endpoint. G1/G2 continuity select in SketchPalette (G2
           uses same cubic formula for now — full quintic G2 is future work).
           Toolbar 'Blend Curve' item in sketchModifyMenuItems. findBlendEndpoint
           + sampleCubicBezier helpers in sketchInteraction/helpers.ts.

[ ] D45. Include 3D Geometry
         - Distinct from Project: pulls 3D edges / work geometry / sketch
           curves into the active sketch as-is (no projection onto plane).
         - Needed for 3D sketches + sweep paths.

[ ] D46. Project to Surface
         - Project sketch curves, edges, vertices, or work points onto a

[x] D47. Sketch Chamfer — 3 variants
         - Shipped: 'sketch-chamfer-equal', 'sketch-chamfer-two-dist',
           'sketch-chamfer-dist-angle' Tool types added. Same corner-finder
           as Sketch Fillet. Equal: both setbacks = dist1. Two-Dist: sb0=dist1
           sb1=dist2. Dist+Angle: sb0=dist1, sb1=dist1*tan(angle). Trims both
           lines to setback points; inserts a new line between them. Dist/Dist2/
           Angle inputs in SketchPalette (visible per variant). Three menu items
           in sketchModifyMenuItems + Chamfer ribbon button. Toolbar wired.

[ ] D48. Arc Length dimension
         - Missing from D28's dimension list. Pick an arc → dimension becomes
           its arc length rather than radius.

[ ] D49. Aligned (true-length) dimension
         - D28 lists linear/angular/radial/diameter but misses Aligned: pick
           2 points and the dimension is measured along the line between
           them, not along X or Y axes.

[ ] D50. AutoConstrain
         - Fusion's SKT-AUTO-CONSTRAIN: analyzes the current sketch, proposes
           constraints + dimensions in purple, lets user tweak via a slider
           (how aggressive) and selectively apply. Depends on the solver from
           D27 but also needs a candidate-generation pass (near-parallel,
           near-horizontal, near-equal-length detection with tolerances).

[ ] D51. Curvature constraint (G2)
         - Not in D29's list. Constrains a spline and another sketch curve
           to share curvature (smooth continuous G2 blend). Only meaningful
           once splines (D9) ship.

[ ] D52. Symmetry constraint (explicit)
         - Already listed as a geometric constraint in D29 but Fusion's
           Mirror tool AUTO-APPLIES symmetry. When D21 (sketch mirror) lands,
           every mirrored pair must get a symmetry constraint so dragging
           one side drags the other. Track as a separate acceptance check.
