FORM WORKSPACE — MODIFY STUBS (blocked by missing pickers/solvers)
═══════════════════════════════════════════════════════════════
All Create and basic Modify tools (Box/Plane/Cylinder/Sphere/Torus/Quadball/Pipe/Face/
Extrude/Revolve/Sweep/Loft, EditForm vertex drag, Subdivide, Crease/Uncrease, Freeze,
Delete) are real implementations in FormInteraction.tsx + SubdivisionEngine. The eleven
tools below are blocked in FormInteraction.tsx because they require pickers or solvers
that do not yet exist.

[ ] FM1. Insert Edge stub — form-insert-edge sets status "requires face-picker (blocked)";
         click handler is a no-op. Needs a face-picker to identify which face to split and
         a SubdivisionEngine.insertEdge() that adds a new edge loop across the face. (hard)
[ ] FM2. Insert Point stub — form-insert-point sets status "requires edge-picker (blocked)".
         Needs an edge-picker + SubdivisionEngine.insertPoint() to subdivide the picked
         edge at the parametric click position. (medium-hard)
[ ] FM3. Bridge stub — form-bridge sets status "requires edge-loop picker (blocked)".
         Needs a two-step edge-loop picker + SubdivisionEngine.bridge() to generate quad
         faces connecting two selected open boundary rings. (hard)
[ ] FM4. Fill Hole stub — form-fill-hole sets status "requires boundary-edge picker
         (blocked)". Needs a boundary-edge picker + SubdivisionEngine.fillHole() to cap
         an open loop with a planar or minimal-surface patch. (medium-hard)
[ ] FM5. Weld stub — form-weld sets status "requires multi-vertex picker (blocked)".
         Needs multi-vertex selection + SubdivisionEngine.weld() to merge coincident or
         near-coincident vertices into a single vertex. (medium)
[ ] FM6. Unweld stub — form-unweld sets status "requires vertex-picker (blocked)".
         Needs vertex-picker + SubdivisionEngine.unweld() to split a shared vertex into
         per-face copies, opening a seam. (medium)
[ ] FM7. Flatten stub — form-flatten sets status "requires plane-projection solver
         (blocked)". Needs selection of vertices + a target plane, then projects each
         vertex onto that plane while holding edge lengths. (medium-hard)
[ ] FM8. Make Uniform stub — form-uniform sets status "requires length-equalizer
         (blocked)". Needs SubdivisionEngine.makeUniform() to equalize edge lengths
         across the cage via iterative Laplacian smoothing of edge midpoints. (medium-hard)
[ ] FM9. Pull stub — form-pull sets status "requires limit-surface solver (blocked)".
         Needs SubdivisionEngine.pullToLimitSurface() to move cage vertices toward the
         Catmull-Clark limit positions (approximating a smooth surface fit). (hard)
[ ] FM10. Interpolate stub — form-interpolate sets status "requires interpolation solver
          (blocked)". Needs a through-point interpolation system: fit a control cage such
          that the smooth limit surface passes through given world-space points. (very hard)
[ ] FM11. Thicken Form stub — form-thicken sets status "requires offset-cage engine
          (blocked)". Needs SubdivisionEngine.thickenCage() to offset each cage vertex
          outward along its averaged normal, generating a double-sided shell cage body. (hard)
