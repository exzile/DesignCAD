## Slicer engine bugs fixed (post-completion audit)

1. Brim offset direction — was inward, fixed to outward (negative offset)
2. Support emission inside model — break instead of flag, fixed with boolean
3. offsetContour spike lines — added maxReach clamp (10× offset + 1mm)
4. Raft top layer default — ?? 1 → ?? 2 to match UI default
5. Grid infill rotation — was fixed 0°/90°, now alternates with 45° offset per layer
6. Triangle infill rotation — was fixed angles, now cycles 3 rotations per layer
7. Fan off at end — M107 was not emitted before end G-code
8. Cubic infill — was emitting only 1 angle/layer, now emits all 3 interlocked sets
9. infillOverlap not applied — regular infill region was not expanded by infillOverlap %
10. supportSpeed hardcoded — was pp.printSpeed * 0.8, now uses pp.supportSpeed with fallback
